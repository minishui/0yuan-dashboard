率类指标计算逻辑（全部按 SUM(分子)/SUM(分母) 加权计算）

指标计算公式分子分母
转化率SUM(转低人次) / SUM(在班人次)转低人次在班人次
端端转化率SUM(转正人次) / SUM(在班人次)转正人次在班人次
端端转化率(数学)SUM(创新思维低转正人次) / SUM(在班人次)创新思维低转正人次在班人次
端端转化率(阅读)SUM(阅读低转正人次) / SUM(在班人次)阅读低转正人次在班人次
端端转化率(英语)SUM(英语低转正人次) / SUM(在班人次)英语低转正人次在班人次
出勤率(任一讲)SUM(任一讲出勤人数) / SUM(在班人次)任一讲出勤人数在班人次
完课率(任一讲)SUM(任一讲完课人次) / SUM(在班人次)任一讲完课人次在班人次
加微率SUM(加微人次) / SUM(在班人次)加微人次在班人次
APP登录率SUM(app登录人次) / SUM(在班人次)app登录人次在班人次
各讲出勤率SUM(各讲出勤人数) / SUM(在班人次)一~五讲出勤人数在班人次
各讲完课率SUM(各讲完课人数) / SUM(在班人次)一~五讲完课人数在班人次
任一讲出勤转化率SUM(转低人次) / SUM(任一讲出勤人数)转低人次任一讲出勤人数

GMV 计算逻辑

指标计算公式单价规则
低价GMVSUM(转低人次 × 单价) 按期次累加春6-春9期 = 169元/人；春10期及以后 = 153元/人
正价GMV(数学)SUM(创新思维低转正人次) × 2899固定单价 2,899元/人
正价GMV(英语)SUM(英语低转正人次) × 2899固定单价 2,899元/人
正价GMV(阅读)SUM(阅读低转正人次) × 2599固定单价 2,599元/人
总GMV低价GMV + 正价GMV(数学) + 正价GMV(英语) + 正价GMV(阅读)-
